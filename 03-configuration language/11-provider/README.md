# Provider

https://developer.hashicorp.com/terraform/language/meta-arguments/resource-provider

O meta-argumento provider no Terraform é usado para especificar explicitamente qual provedor deve ser usado para um determinado recurso, módulo ou data source. Isso é particularmente útil em cenários onde você está usando múltiplos provedores do mesmo tipo, mas configurados de maneira diferente, como diferentes contas AWS, diferentes regiões, ou diferentes endpoints de API.
